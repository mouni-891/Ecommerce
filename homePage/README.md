# Ecommerce Project
🛍️ Ecommerce Store

A responsive Ecommerce website built with HTML, CSS, and JavaScript.
Features product listing, search, filtering, sorting, and a shopping cart system.

✨ Features

Product listing with price, discount, rating, and reviews

Search products in real-time

Filter by category (Electronics, Accessories)

Sort products (Price low→high, high→low, newest, most popular)

Shopping cart modal with quantity control and remove items

Responsive design for desktop & mobile

Dark mode support

🛠️ Technologies Used

HTML5 — structure

CSS3 — styling, responsive layout, animations

JavaScript — dynamic product rendering, cart logic, filters

📂 Project Structure
ecommerce/
│── index.html       # Main layout
│── style.css        # Styling
│── script.js        # Product logic & cart system
│── README.md        # Project documentation

🚀 How to Run

Clone the repository:

git clone https://github.com/mouni-891/Ecommerce.git


Open the folder

Open index.html in your browser

No server required — works locally.
